
# 🤣🚨🚨 Proyecto de Gestión Académica - AAUCA

## Universidad Afroamericana de África Central (AAUCA)

Este proyecto tiene como finalidad la implementación de un **Sistema Integral de Gestión Académica** para la Universidad Afroamericana de África Central (AAUCA), orientado a mejorar los procesos administrativos, académicos y de seguridad informática en las cuatro facultades que conforman la institución.

---

## ➕ Objetivos del Proyecto
- Digitalizar y optimizar procesos administrativos y académicos.
- Reducir en más del 50% los tiempos de inscripción, gestión de notas y generación de reportes.
- Garantizar la seguridad de la información mediante cortafuegos, IDS/IPS y monitoreo de red.
- Aprovechar el talento interno de estudiantes y egresados en el desarrollo del sistema.
- Asegurar sostenibilidad institucional sin depender de licencias costosas o proveedores externos.
- Posicionar a AAUCA como referente en digitalización universitaria en África Central.

---

## 🔍 Arquitectura del Sistema
El sistema se estructura en módulos, iniciando con el **Módulo de Gestión de Departamento**.

### Back-end
- Lenguaje: **PHP** (con PDO para acceso seguro a bases de datos)
- Base de datos: **MySQL**
- Arquitectura: **MVC (Modelo-Vista-Controlador)**
- Servicios: **REST API** para comunicación con el front-end y futuras integraciones

### Front-end
- **HTML5 / CSS3 / Bootstrap**: interfaz moderna y adaptable
- **JavaScript / jQuery / AJAX**: interactividad y comunicación asíncrona
- **Chart.js / D3.js**: visualización de datos académicos

### Infraestructura
- **Apache**: servidor web confiable
- **Git/GitHub**: control de versiones y colaboración
- **UML**: diagramas de casos de uso, clases y secuencia para documentar la arquitectura

---

## 📂 Modelo de Datos (Resumen de Entidades)
- **Usuario**: login, password, rol, estado
- **Profesor**: datos personales, departamento, responsabilidades
- **Formación**: institución, tipo, título, nivel
- **Asignatura, Curso, Semestre, PlanEstudio**
- **Estudiante**: datos personales, asignaturas, becas, inscripciones
- **Evaluación y Examen**
- **Noticias, Documentos, Informes, Guías Didácticas**
- **Aulas, Horarios, Clases**
- **Consultas y Destinatarios**
- **Facultades y Carreras**
- **Pagos e Inscripciones**
- **Familiares y Becarios**
- **Static y Contactos**
